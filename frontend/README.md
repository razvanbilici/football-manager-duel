# Football Frontend

Frontend Vue 3 + Tailwind CSS pentru backend-ul Spring Boot.

## Cerințe

- Node.js 18+
- Backend-ul rulează pe `http://localhost:8080`

## Pornire

```bash
cd football-frontend
npm install
npm run dev
```

Aplicația pornește pe `http://localhost:5173`.

## Funcționalități

- **Autentificare** — înregistrare și login (JWT)
- **Jucători** — listare, căutare, cumpărare
- **Cluburi** — vizualizare cluburi și loturi
- **Echipa mea** — formație, tactică, adăugare jucători, publicare
- **Comunitate** — echipe publicate + voturi
- **Transferuri** — istoric transferuri recente
